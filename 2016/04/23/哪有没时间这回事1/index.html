哪有没时间这回事（一）

发表于 2016-04-23   |   分类于 Grow   |     |     阅读次数

一直对时间管理有所关注，利用上下班地铁上时间读完了《哪有没时间这回事》，现在来818这本书——碎片化时代的正确打开方式。

导语：碎片化时代的碎碎念

很多事情不得不做；
很多事情想做但从未去做；
很多事情做了但坚持不下来；
总感觉自己很忙碌，但想不起来到底干了什么；
大量的时间被他人占用，剩下的时间被自己浪费；
早上起不来，晚上不想睡，缺乏锻炼，没空学习，做事效率低，生活不规律；
一堆臭毛病想改改不掉，别人身上的好习惯想学做不到； 

正确的打开方式：“三阶技能”，“三个清单”。
  • 初级技能：需要变换全新视角，重新感受整理被揉碎的时间，尽可能把被碎片占据的－ 时间找回来。 （时间段清单）
  • 进阶技能：需要用争取到到时间丰富自己到技能，建立更广泛到人脉。同时，建立一套良性运转的任务处理系统，高效做事。（碎片清单和每日任务清单）
  • 高级技能：需要把技能植入体内，成为自己的习惯，用全新的自己应对未来。（实现以上的建设性方法；练习正确的步骤而不是结果） 
先写个例子看看是否你需要继续往下读：（感受以下三组标题的区别）
“让1天变28个小时”            “让1天变28个小时的初级技能”
“不用专注也能完成80%任务”     “不用专注也能完成80%任务的进阶技能”
“让高效毫不费力”             “让高效毫不费力的高级技能”

Level 1: 让1天变28个小时的初级技能

1. 从安排日程开始，也许就错了

  • 感受碎片，整理心情。
  • 儿时被动按照课表规划，现在独立自主，处理事情与日俱增，时间越来越碎。
  • 碎了就碎了，坦然接受。
  • 放空大脑，专注当下事务，其余的事情都丢进清单。
  • 列出日常清单，避免焦虑，解放大脑。
  • 有利于统筹日常事务。
  • 列出日常事务清单，不加时间标签，只是单纯的事件罗列。

2. 日程表不靠谱，换时间段试试

  • “严格日程表”，容易被意外事件轻易摧毁。
  • 日程表经常从早起开始就失败了，然后整天将处在挣扎沮丧的状态。
  • 时间表只会逼死你
  • 审视时间、地点、物品、环境、他人和自己的身心状态六大要素。
  • 按照上一条，将一天划分成若干时间段。
  • 可以帮你理解，对自己不切实际的严苛要求是导致事情失败的原因。
  • 根据六大要素安排日常事务。

3. 不断改进的是规划，墙上一挂是鬼话

  • 对的时间做对的事
  • 不断优化每个时间段
  • 用清单整理碎片，事情再多，都是纸老虎，列出来就可以指点江山
  • 划分时间段，帮你划出界限。
  • 碎片化专注：每个时间段内，切换到沉浸模式。
  • 时间段不要安排太满，以便划分出清晰界限。
  • 类似开始工作前我们会泡一杯茶，时间段间状态切换可以用一些仪式性动作来完成。
  • 构建自己的时间段清单，安排每天事宜。（1⃣️明确要做的事。2️⃣把他们分别放入规划好的时间段。3⃣️列出每个时间段的动作清单。）

4. 调整早晚时间，不再瞎忙活

  • 早起：根生蒂固的思维陷阱：“困”是个问题，早起是造成困的原因。
  • 早起：困不是问题，困的时候挺着不休息才是问题。
  • 早起：早起倒逼早睡。起的爽快，睡的心安理得。
  • 早起：正确的时间做正确的事情，错开每天杂事高峰，获得更多时间，更好质量，聚焦时间段安排，优化切实可行方案。
  • 晚间：到点就走，保护自己的晚间。很多时候被加班，那就先建立早起加班的自信。
  • 晚间：加班要早上加，最重要的是不精神加班。
  • 晚间：人的差别在于业余时间，而一个人的命运决定于晚上8点到10点。
  • 晚间：停下白天的快节奏，陪伴家人，准备第二天最重要的事，有勇气去早睡。

总结

  1. 梳理每天的日常事务，因为它们是构成你每一天的基础。
  2. 把自己的一天划分成若干时间段来安排这些日常事务。
  3. 划分时间段的过程是需要持续进行的，持续地稳定六大要素。
  4. 通过建立良好地作息规律来优化时间段，早起可能是改头换面的起点。
  5. 重新审视晚间时光的意义，并通过建立良好的程序来管理自己。
